# shared-plugin-interfaces

#### 项目介绍
Share plugin interface for openkm plugin system

#### 软件架构
此为Openkm通用插件接口，需要在本地maven库安装才能使用

#### 安装教程

```bash
./build.sh
```

#### 使用说明

在项目pom.xml中添加下面代码

```xml
<dependency>
    <groupId>space.terwer</groupId>
    <artifactId>shared-plugin-interfaces</artifactId>
    <version>4.1.14</version>
</dependency>
```
