# openkm-common

common libraries for openkm

## Install

### Bash

```bash
./build.sh
```

### PowerShell

```bash
pwsh ./build.ps1
```

## Uasge

Add the following code in `pom.xml`

```xml

<dependency>
    <groupId>space.terwer</groupId>
    <artifactId>openkm-common</artifactId>
    <version>5.0.1</version>
</dependency>
```
